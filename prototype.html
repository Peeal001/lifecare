Find Nearby Clinics

Users can search for free healthcare services by location using zip code or GPS. The interface shows a map and list of available clinics.

Eligibility Checker

A simple, friendly quiz helps users determine if they qualify for free healthcare based on their location, insurance status, and immigration status.

Appointment Booking

After selecting a clinic, users can book appointments in just two steps - choosing a time slot and providing required information.

Key Design Considerations

Privacy Protection: We designed the app to collect minimal personal information and clearly communicate data usage policies.

Language Support: The interface supports multiple languages with clear iconography to overcome language barriers.

Simplified Navigation: The step-by-step flow guides users through the process with minimal complexity.
